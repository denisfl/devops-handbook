# Junior DevOps: Zero to Hero

Семидневный интенсив для погружения в профессию инженера эксплуатации.
Формат: конспекты лекций + практические задания.

---

### [День 1. Основы Linux и Bash](day1-linux-bash.md)
*Фундамент*. Идеология Unix, файловая система, права доступа (chmod/chown), SSH и базовая автоматизация на Bash. Перестаем бояться терминала.

### [День 2. Git и Systemd](day2-git-systemd.md)
*Инструменты*. Система контроля версий: объекты, ветвление, решение конфликтов. Systemd: как запускать приложения в фоне, управлять зависимостями и логами (journalctl).

### [День 3. Docker и Docker Compose](day3-docker-compose.md)
*Контейнеризация*. Изоляция приложений. Образы vs Контейнеры. Слои (Layers). Оркестрация локального окружения с помощью Compose.

### [День 4. Сети и Масштабирование](day4-networking-scaling.md)
*Транспорт*. Как работают TCP/IP и DNS. Модель OSI. Балансировка нагрузки (Load Balancing) и проксирование через Nginx.

### [День 5. Python для автоматизации](day5-python-scripting.md)
*Логика*. Почему Bash недостаточно. Структуры данных Python, работа с внешними API (`requests`), обработка ошибок и парсинг логов.

### [День 6. Мониторинг и Стабильность](day6-monitoring-stability.md)
*Наблюдаемость*. "Золотые сигналы" мониторинга. Логи, метрики, алертинг. Стратегии деплоя (Blue/Green) и резервное копирование.

### [День 7. Архитектура и Будущее](day7-microservices-next.md)
*Системный дизайн*. Монолит против Микросервисов. CAP-теорема. Идемпотентность. 12 Factor App. Карта развития (Roadmap) до уровня Middle.
